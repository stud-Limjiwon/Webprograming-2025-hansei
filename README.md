## 🛠️ 프론트엔드 구현 특징

* **HTML:** WAI-ARIA 및 의미론적 마크업 준수
* **CSS:** 고대비/대형 폰트 등 접근성 테마 지원 및 반응형 디자인 적용
* **JavaScript:** 개인 설정 저장/로드 로직 및 IoT 통신(가정) 기반 UI 전환 로직 구현
